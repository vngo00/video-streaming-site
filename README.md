A platform that allows users to upload, view, and share videos with others. Users can upload their own videos on a wide range of topics, from personal vlogs to educational content, entertainment, or tutorials. These videos can be accessed by a global audience, and users often interact through likes, comments, and shares.
## Build Database
create .env file and fill in the required environment variables:
DB_HOST, DB_NAME, DB_USER, DB_PASSPWORD, PORT
## Build Instructions
1. cd video-stream-site/application
2. npm install

## Run Instructions
1. npm start
